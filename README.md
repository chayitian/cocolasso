# cocolasso
参考 BDCoCoLasso 作者的 R 语言包实现 Python 的 CoCoLasso 并进行复现，之后拓展到 AdaCoCoLasso（自适应CoCoLasso）并进行复现。<br>
BDCoCoLasso 作者的 R 语言包地址：https://github.com/celiaescribe/BDcocolasso.git <br>
参考文献：<br>
【1】Datta A , Zou H .CoCoLasso for High-dimensional Error-in-variables Regression[J].Annals of Statistics, 2016, 45(6):2400-2426.<br>
【2】Yu Q .Adaptive CoCoLasso for High-Dimensional Measurement Error Models[J].Entropy, 2025, 27(2).<br>
